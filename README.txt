Playing the game:
To play the game, download the Unity Build zip folder in releases, extract the folder and run the exe

Controls:
W - Forward
A - Backward
S - Walk Left
D - Walk Right

Left Click - Grab/Throw telekinetic object (grab when hovering over object, throw when object is pulled)
Right Click - Create shield debris (only on shield spawnable surfaces, e.g: the floor)
Space (when NOT grounded) - Activate/Deactivate levitation
Tab - Swap camera position
